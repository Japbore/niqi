# Niqi — Requisitos v0.2.0

Esta versión añade mejoras significativas de usabilidad, gestión de cantidades, y una reorganización de la interfaz hacia un estilo "App Móvil".

## Historias de Usuario

| ID | Como... | Quiero... | Para... |
|---|---|---|---|
| HU-10 | usuario | poder editar un producto ya añadido | corregir errores en el nombre o cambiar su categoría |
| HU-11 | usuario | definir la cantidad y unidades de un producto | saber exactamente cuánto debo comprar de cada cosa |
| HU-12 | usuario | importar usando un formato natural (ej: "2kg Patatas,Frescos") | pegar listas de la compra con cantidades integradas rápidamente |
| HU-13 | usuario | tener un menú superior compacto y desplegable | ver la app más limpia y acceder a opciones secundarias (Importar, Acerca de...) fácilmente |
| HU-14 | usuario | localizar productos rápidamente filtrando texto | encontrar artículos en listas muy largas |
| HU-15 | usuario | tener la interfaz en modo oscuro | utilizar la aplicación sin forzar la vista en entornos de poca luz |

## Requisitos Funcionales

| ID | Requisito | HU relacionada |
|---|---|---|
| RF-13 | **Edición de producto**: Al pulsar sobre el texto de un producto, se permite editar su nombre, cantidad y categoría. | HU-10 |
| RF-14 | **Cantidades**: El modelo de datos y la UI soportan volumen/unidades, visible junto al nombre. | HU-11 |
| RF-15 | **Parseo Natural (Import.)**: El texto de entrada sigue la regla opcional `[Cantidad ]Nombre[,Categoría]` (ej: `2 gominolas`, `1kg de naranjas,Fruta`). | HU-12 |
| RF-16 | **Navbar App**: Banda de color superior compacta. Logo+Nombre a la izquierda, botón "Hamburguesa" a la derecha. | HU-13 |
| RF-17 | **Opciones del Menú**: El menú contiene "Importar" y "Acerca de". La acción "Acerca de" abre un modal descriptivo del proyecto. | HU-13 |
| RF-18 | **Filtrado en vivo**: Campo "Buscar" sobre la lista. Oculta items de acuerdo al nombre desde el 3er carácter ingresado. Incluye botón "Borrar" (X). | HU-14 |
| RF-19 | **Modo Oscuro**: Implementación nativa con CSS automático basado en las preferencias del sistema y opción de conmutador (opcional). | HU-15 |
